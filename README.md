**WeHire** is an AI-powered career development platform designed to help students and professionals prepare for every stage of their job journey. The platform offers intelligent **resume analysis** with personalized feedback and ATS optimization, realistic **AI-powered mock interviews** with performance insights, and a dedicated **career assistant** that provides skill-gap analysis, learning recommendations, and personalized career roadmaps. It also includes an **AI writing assistant** to help users create professional resumes, cover letters, emails, and LinkedIn summaries. Built with **Flutter**, **Spring Boot**, **Firebase Authentication**, and **Gemini AI**, WeHire delivers a seamless, secure, and scalable experience, making career preparation smarter, faster, and more personalized.
